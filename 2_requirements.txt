Installation of the requirements: 

1.create a text file and rename it as requirements.txt .
2.Now add the below contents into the file and save the file.

numpy
opencv-python
tensorflow
scikit-image
matplotlib
jupyter

3.Open the command prompt,type the below code and hit enter.
   
     pip install -r requirements.txt



Creating a image folder:(For encryption purpose)
 create a folder named images and save atleast 5 images with high resolution, so that ai selects best image for encryption purpose.
